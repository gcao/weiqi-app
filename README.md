# Weiqi App

## Usage

* Build assets

<code>
node_modules/webpack/bin/webpack.js -w
</code>

* Run the server

<code>
cd build && python -m SimpleHTTPServer 9000
</code>

* Access the application

<code>
open http://localhost:9000
</code>
